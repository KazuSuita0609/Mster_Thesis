**著者：** 吹田一馬

> 本リポジトリは，卒業論文（淀川流域における時空間降水パターンのクラスター分析）の成果をベースとした修士論文研究のコードおよびドキュメントを管理する。

---

## 研究背景（卒業論文から）

卒業論文では，d4PDF 5kmグリッド地域気候モデルデータを用いて淀川流域の降水イベントを抽出し，複数のクラスタリング手法（K-means，SOM，Autoencoder，DEC，Convolutional AE）を比較した．現在気候（1950–2010年）と +4K 昇温将来気候シナリオにおけるパターン分布の変化を統計的に検定し，一部の降水パターン（南部集中型）の出現頻度が将来気候下で増加する傾向を示した．

**卒業論文コード：** [`../ProgramCode_for_graduation_thesis/`](../ProgramCode_for_graduation_thesis/)

## 使用データ

| 項目 | 内容 |
|---|---|
| データセット | d4PDF 5kmグリッド地域気候モデル（DS2022） |
| 対象流域 | 淀川流域（緯度：34.3–35.6°N，経度：135.3–136.4°E） |
| 時間解像度 | 1時間 |
| アンサンブルメンバー数 | 12メンバー（m001–m012） |
| 現在気候期間 | 1950–2010年 |
| 将来気候 | +4K 昇温実験 |

データの詳細は [DS2022 マニュアル](https://diasjp.net/ds2022/manual_chapter1.html#main1) を参照。
